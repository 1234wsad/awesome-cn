[Prisma](https://www.prisma.io/ 'Build a GraphQL server with any database')  是一个高性能的开源GraphQL ORM层，在您的GraphQL服务器中执行繁重的工作.  它将您的数据库转换为GraphQL API，您的解析器可以通过GraphQL绑定使用它.
